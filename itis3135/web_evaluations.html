Web Evaluations

Good "CRAP" Site example

Mondrian and me. - visit site -

Description: Mondrian and Me is an interactive web page that allows users to explore abstract artistry through geometric shapes. The design is inspired by Piet Mondrian, making it visually engaging and creative. The site encourages users to have fun while procrastinating, blending entertainment with an artistic twist.

CRAP Analysis

  • Contrast: The site uses bold primary colors (blue, red, and white) with sharp black lines, creating clear contrast that makes the geometric shapes stand out.
  • Repetition: The repeated use of geometric squares and color patterns provides a consistent, balanced visual experience, reflecting Mondrian's style.
  • Alignment: The geometric grid layout is perfectly aligned, giving the site a polished, modern aesthetic that draws attention to the interactive elements.
  • Proximity: Interactive elements are grouped logically, allowing users to easily explore different parts of the canvas and engage with the art.
